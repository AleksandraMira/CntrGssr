# CntrGssr
Guess a country mobile game
## Introduction
This is the final project for Coroutines Mastery course by kt.academy.
## Project Overview
CntrGssr is a simple game where you need to guess a country based on hints. The project consists of application code and 3 example unit tests.
## Game Rules
You have 3 chances to guess the name of the country and 3 hints you can use. The final score is calculated based on used hints and unsuccessful tries.
## Hints
You have 3 types of hints:
- Continent name
- Capital name length (only letters count)
- Country population
